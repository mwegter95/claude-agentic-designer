# Transformation Playbook — Element by Element

Use this in Phases 4–5 to convert a non-branded slide into an on-brand one. Work
top-down: structure first (layout + background), then content into placeholders,
then type and color, then components, then logo and signature features, then
alignment. Each step references the **Design Contract** built in Phase 1.

## Step 0 — Choose the master layout
- Match the slide's **intent** to a named master layout:
  - single big idea / opener → **Title** or **Hero**
  - new section → **Section Divider**
  - prose + bullets → **Content** (one or two column)
  - metrics → **Stat / Callout**
  - pull-quote → **Quote**
  - photo-led → **Image / Full-bleed**
  - side-by-side → **Comparison / Two-column**
  - end → **Closing**
- Apply that layout to the slide. Prefer a named match (a layout literally named
  "Section Header" for a section divider). If nothing fits, pick the closest and
  note the compromise in the report — never invent a new layout.

## Step 1 — Background
- Set the layout's correct background (solid/gradient/image) from the contract.
- Restore any baked-in background graphics/accent shapes the layout expects.
- Pick the light or dark variant to suit the content and logo.

## Step 2 — Content into placeholders
- Move each piece of text/image out of free-drawn boxes and into the layout's
  **placeholders** (title → title placeholder, body → body placeholder, etc.).
- Delete now-redundant manual text boxes and stray shapes.
- Keep the wording intact (preserve meaning); only reflow to fit.

## Step 3 — Typography
For every text level, apply the contract's type scale:
- Font family (heading font for titles, body font for body).
- Point size, weight, casing, letter-spacing, line spacing for that level.
- Text color by **palette role** (e.g. title = Text 1, kicker = Accent 1).
- Bullet glyphs and indents per level; convert ad-hoc bullets to the brand style.
- Remove any leftover non-brand fonts entirely — no exceptions.

## Step 4 — Color
- Recolor every element by its **role**, pulling exact theme colors:
  - titles/body → text colors; accents/dividers/bars → accent colors;
    fills/backgrounds → background colors; links → hyperlink color.
- Replace each off-palette hex with the nearest brand color **for that role** (match
  by function, not just by closest RGB).
- Apply the brand's tints/shades where the master uses them (e.g. a 20% accent fill).

## Step 5 — Components & shapes
- Restyle shapes to the default component style: corner radius, fill, line, shadow.
- Convert generic stat/number blocks to the master's **callout** style.
- Replace ad-hoc rules/lines with the brand **divider / accent bar** (right color,
  thickness, length).
- Normalize icons to the brand icon style (line vs. filled, stroke weight, color).
- Apply the brand **image treatment**: corner radius, framing/full-bleed, duotone or
  overlay, and consistent shadow.
- Restyle tables and charts to the brand table/chart style and series colors.

## Step 6 — Logo
- Place the correct **logo variant** for the background (reversed on dark, primary on
  light) at the contract's position, size, and clear-space.
- On slide types that should carry a logo (title/closing/footer), ensure it's present;
  on those that shouldn't, ensure it's absent.

## Step 7 — Signature features
Recreate the small brand motifs from the contract's signature-features list:
- Accent rule/underline under the title (exact color, thickness, offset).
- Footer text + divider; page number in the brand style/position.
- Section color-coding if the deck uses it.
- Any repeated decorative accent (corner wedge, dotted rule, brand pattern).
- Kicker/label-above-title pairing if that's the brand pattern.

## Step 8 — Alignment, spacing, density
- Snap elements to the grid; equalize margins and gutters.
- Enforce the density rules (max bullets/slide, words/bullet); if the slide is
  overloaded, reflow or split it and note the split.
- Balance whitespace to match the examples' airiness; align to the baseline.

## Keep / Restyle / Move / Rebuild — quick rule
- **Keep** anything already on-brand and correctly placed.
- **Restyle** on-brand-able elements (text, native shapes) in place.
- **Move** content into the right placeholder/position.
- **Rebuild** only what can't be restyled (off-brand free-drawn graphics replaced by
  brand components).
