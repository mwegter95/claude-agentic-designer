# Self-Review Rubric

Use this in Phase 6. Score the transformed slide against each criterion as a fresh
critic. Any **hard-fail** criterion that misses must be fixed before you finish,
regardless of how good the rest looks. Iterate up to 3 times, then ship the best
version with any residual issues noted in the report.

## Criteria

| # | Criterion | Type | Pass bar | What to check |
|---|-----------|------|----------|----------------|
| 1 | **Fonts on brand** | hard-fail | All text uses only the master's approved typefaces | No stray Calibri/Arial or non-brand fonts at any text level |
| 2 | **Colors on brand** | hard-fail | Every color is a theme color (or its brand tint) | No off-palette hex; colors applied by correct role |
| 3 | **Layout from master** | hard-fail | The slide uses a real named master layout | Content sits in placeholders, not free-drawn boxes |
| 4 | **Logo correctness** | hard-fail | Correct variant, position, size, clear-space; present/absent per slide type | Reversed logo on dark, primary on light; not stretched |
| 5 | **Density within rules** | scored | No slide exceeds the master's bullet/word limits | Reflowed or split if overloaded |
| 6 | **Visual hierarchy** | scored | Clear focal point; title dominant | Matches the examples' emphasis pattern |
| 7 | **Whitespace & alignment** | scored | Comfortable density; consistent margins/gutters; grid-aligned | No crowding, no ragged alignment |
| 8 | **Fidelity to examples** | scored | Looks like it belongs in the example set | Same taste in imagery, spacing, hierarchy |
| 9 | **Subtle-feature match** | scored | Signature motifs recreated | Title accent rule, footer, page number, section coding, image radius, icon stroke |
| 10 | **Content fidelity** | hard-fail | The slide's information is intact | Nothing dropped/distorted while restyling (unless user asked) |

## How to score
- For each criterion, decide pass/fix and cite specific evidence (which element,
  which attribute).
- Fix every hard-fail miss and every scored item that's clearly weak.
- Re-review after fixes. Stop when all hard-fails pass and scored items are solid, or
  after 3 iterations.

## Finish gate
- **Ship** when: criteria 1–4 and 10 pass, and 5–9 are solid.
- If you must ship with a residual issue (e.g. a slide you recommend splitting for
  density, or missing content that needs a user-provided source), state it explicitly
  in the Phase 7 report rather than silently leaving it.
